# 📚 Educational OTP/Call API Demo

## ⚠️ IMPORTANT - READ FIRST

**This is an EDUCATIONAL DEMO ONLY**

- ❌ NO real SMS/Calls are sent
- ❌ NO actual API calls are made
- ✅ Only shows API STRUCTURE for learning
- ✅ Includes legal warnings and consent system

## 🎯 Purpose

This demo helps students understand:
- How SMS/OTP APIs are structured
- Different HTTP methods (POST/GET)
- Request/Response formats (JSON, form-data)
- Importance of API keys and authentication
- Legal and ethical considerations

## 🚫 What This DOES NOT Do

- Does NOT send any real SMS
- Does NOT make any real calls
- Does NOT store any phone numbers
- Does NOT work with real API keys
- Cannot be used for actual bombing

## ✅ Legal Alternatives for Real SMS

For legitimate SMS/Call needs, use official services:

1. **Twilio** - International SMS/Call API
2. **MSG91** - Indian SMS Gateway
3. **Fast2SMS** - Free trial available
4. **TextLocal** - Bulk SMS service
5. **AWS SNS** - Cloud messaging

## 📜 Legal Notice (India)

Under IT Act 2000:
- Section 66D: Up to 3 years imprisonment
- Section 72: Up to 2 years imprisonment
- Section 43: Compensation up to ₹1 crore
